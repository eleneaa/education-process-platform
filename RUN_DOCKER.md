## 🎯 Что произойдёт при запуске

1. **Docker запустит 3 сервиса:**
   - Frontend (Vite dev сервер) → http://localhost:5173
   - Backend (FastAPI) → http://localhost:8000
   - Database (PostgreSQL) → localhost:5432

2. **При первом посещении http://localhost:5173:**
   - Автоматически откроется окно логина
   - Используйте admin@example.com / changethis
   - Попадёте на главный дашборд

3. **На главном дашборде вы увидите:**
   - Четыре новые красивые карточки метрик
   - Графики аналитики
   - Таблицы с данными

## 🐛 Если что-то не работает

### Frontend не грузится?
```bash
docker-compose logs frontend
# Посмотрите ошибки в логах
```

### Порт 5173 занят?
Измените в `compose.override.yml` или используйте другой:
```bash
docker-compose down
docker-compose up -d
```

### Очистить всё и начать заново?
```bash
docker-compose down -v
docker-compose up -d --build
```
