## 💾 Интеграция с реальными данными

Компоненты готовы работать с данными:

```tsx
import { useQuery } from "@tanstack/react-query"
import { MetricCard } from "@/components/Dashboard"

export function MyDashboard() {
  const { data: metrics } = useQuery({
    queryKey: ["metrics"],
    queryFn: () => fetch("/api/metrics").then(r => r.json()),
  })

  return (
    <MetricCard
      icon={Users}
      label="Студентов"
      value={metrics?.total || 0}
      change={{ value: metrics?.change || 0, trend: "up" }}
      variant="success"
    />
  )
}
```

## ❓ Troubleshooting

**Компоненты не отображаются?**
- Убедитесь что пути импорта правильные: `@/components/Dashboard`
- Проверьте что `@` alias настроен в `tsconfig.json`
- Посмотрите консоль браузера на ошибки

**Стили не применяются?**
- Tailwind CSS должен быть подключен в `src/index.css`
- Убедитесь что `tailwindcss` установлен в package.json
- Очистите кэш браузера (Ctrl+Shift+Delete)

**Dark mode не работает?**
- ThemeProvider должен обёртывать приложение в `src/main.tsx`
- Проверьте что `next-themes` установлен
